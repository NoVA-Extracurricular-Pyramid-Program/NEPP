Set up your Firestore database collections

What this does:

This tool initializes your Firestore database with the required collections for NEPP to function properly.

Collections that will be created:

  • folders - For the file organizer system
  • resources - For uploaded files and documents
  • users - For user profiles and authentication
  • groups - For team/group management
  • forms - For forms and surveys
  • announcements - For announcements system
  • events - For events management
  • form_responses - For form submissions
  • feedback - For user feedback
